## ИИММ: лабораторая работа 1

Задание состояло в реализации простой программы для трекинга объектов.

Некоторые подробности реализации:
- раз в три кадра обновляется таймер на видео;
- раз в три кадра ставим новую метку, а именно: применяем пороговую обработку к изображению с помощью cv2.inRange() в HSV-диапазоне (90, 75, 75) - (150, 200, 200) в соответствии с задачей трекинга персонажа с одеждой синих тонов;
- каждый кадр рисуем предыдущее значение таймера, а также абсолютно все накопленные центроиды.

### Действия для запуска

Если необходимо, склонируйте репозиторий:
```
git clone https://git.miem.hse.ru/nsvasileva_31/aimm.git aimm_nsvasileva_3
cd aimm_nsvasileva_3
```

Переключитесь на ветку первой лабораторной работы:
```
git checkout lab1
```

Создайте окружение и установите зависимости:
```
python -m venv venv

// Для Windows
venv\Scripts\activate

// Для Linux
source venv/bin/activate

pip3 install -r requirements.txt
```

Далее загрузите с [диска](https://disk.360.yandex.ru/d/KgQeuNCmbpYkDQ) видео для обработки в папку репозитория.

Наконец, запустите:
```
python main.py
```

По выполнении работы программы можно увидеть результат в файле с пометкой tracked.
